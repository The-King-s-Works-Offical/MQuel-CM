# MQuel Content Manager

## Introduction

MQuel Content Manager is a desktop application that allows users to manage content for their MQuel website. The application is built using Electron, a framework for building cross-platform desktop applications.

## Features

MQuel Content Manager includes the following features:

* Create, edit, and delete pages
* Upload and manage images and other files
* Preview pages before publishing
* Publish pages to your MQuel website

## Installation

To install MQuel Content Manager, download the latest release from the [releases page](https://github.com/MQuel/content-manager/releases). Once the download is complete, double-click the installer to begin the installation process.

## Usage

To use MQuel Content Manager, open the application and sign in to your MQuel account. Once you are signed in, you will be able to see a list of all of your pages. To create a new page, click the "Create Page" button. To edit a page, click the "Edit" button next to the page title. To delete a page, click the "Delete" button next to the page title.

To upload an image or other file, click the "Upload" button and select the file you want to upload. Once the file is uploaded, you will be able to insert it into a page by clicking the "Insert Image" or "Insert File" button.

To preview a page, click the "Preview" button. This will open a new window with a preview of the page. To publish a page, click the "Publish" button. This will publish the page to your MQuel website.
